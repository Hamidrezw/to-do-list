# To-Do List App 📝

A **responsive To-Do List application** built with **HTML, CSS, and JavaScript**.  
Manage your daily tasks efficiently with **add and delete functionality**, and keep your tasks even after page refresh using **local storage**.

---

## Features ✨
- Add new tasks  
- Delete tasks  
- Persistent tasks with **local storage**  
- Fully **responsive design** for mobile and desktop
